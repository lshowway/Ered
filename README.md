## Introduction
> This is the code for the paper [Ered: Enhanced Text Representations with Entities and Descriptions](https://arxiv.org/abs/2208.08954).
## Installation
> `pip install -r requirements.txt`  
> `python 3.9.9 cuda 11.6, torch 1.13.1`
>
